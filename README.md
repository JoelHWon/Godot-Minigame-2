In _sprite.modulate = Color(r, g, b), the period is used to declare the type of data. I think this because I've seen it before like in pictures where they say things like "Photo.png" or "Photo.jpg". 
I learned in the documentation that by using "Color", it makes the three numbers in the parentheses values based on RGB.
I also learned that by messing around with the modulate property in the player node, it changes the entire sprite's color. For example, I changed the redness of the sprite and it made it so that the sprite turned really red.
